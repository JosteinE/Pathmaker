<H2 align="center">Building a path</H2>
<br>Build a path simply by holding down shift and righ click on any tile, object or NPC, then click the "Add &lt;selected&gt; to &lt;path&gt;" option to mark it as a path point. Marking the first tile (with 3 or more tiles already marked) will give you the option to loop the path.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/c83562d4-9fa9-4bb1-8db0-c022459ea727"><br>
<H2 align="center">Offset the path</H2>
You can offset the line to be drawn from the tile- or entity- center or by their inner or outer bounds.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/87e02db5-a24b-4963-b505-717692b66ba5"><br>
<H2>Create additional paths</H2>
To create a new path use the in-game, shift-down right-click menu or type in a unique name in the "Active Path" field in the plugin panel. You can quickly switch out the "Active Path" by clicking on any path label.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/9549f317-a52a-4901-9d6f-c5ff894d4d55"><br>
<H2 align="center">Delete points</H2>
Delete a point either by using the in-game, shift-down right-click menu on a previously marked tile or by clicking a gray X in the plugin panel<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/29fbadec-bc88-40ea-afc5-784a33802478"><br>
<H2 align="center">Rearrange points</H2>
Rearrange points by clicking on the up or down arrow next to any point's index or by manually typing one in.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/c12c2de8-5ee1-411e-8c70-7e8f0db3a760"><br>
<H2 align="center">Rearrange paths</H2>
Click and drag any path label in the plugin panel to rearrange the path order.<br>

<H2 align="center">Hide specific lines</H2>
To hide specific lines between points, click on the space between two points in the plugin panel.<br>

<H2 align="center">Set text mode</H2>
Set the text overlay mode for any path by going into the path specific drop-down-options-menu, by clicking the triple vertical dots button then select text.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/cf3316b4-056c-4056-a163-828c9f591672"><br>
<H2 align="center">Set highlight mode</H2>
Set the tile and entity highlight mode for any path by going into the path specific drop-down-options-menu, by clicking the triple vertical dots button then select highlight.<br>
<br><img width="80%" src="https://github.com/user-attachments/assets/d3cab1cd-002b-4458-bfe5-233d0eef6cf9"><br>
<H2 align="center">Export, import & grouping</H2>
- To export a path, first make that path the active path, then click on the button above with the arrow pointing out. This will copy a JSON string to your clipboard, which can be pasted and shared with other players and imported into their client.<br>
- To import a path, have the JSON string be copied to your clipboard then click the import button, the button next to the export button with the arrow pointing down.<br>
- To group paths, drag and drop a path directly on another path or group in the plugin panel.<br>
- To ungroup a path, click and drag a path from within a group and drop the path outside of that group.
